# PDF Variables

First, download some PDFs from arxiv.org:

```bash
npx promptfoo@latest init --example pdf-variables
# or simply
./fetch_pdfs.sh
```

This example is pre-configured in `promptfooconfig.yaml`. That means you can just run:

```bash
npx promptfoo@latest init --example pdf-variables
# or simply
promptfoo eval
```
