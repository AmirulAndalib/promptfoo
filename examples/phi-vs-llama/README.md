To get started:

```bash
npx promptfoo@latest init --example phi-vs-llama
# or simply
ollama pull llama3
ollama pull phi3
```

Next, edit promptfooconfig.yaml.

Then run:

```bash
npx promptfoo@latest init --example phi-vs-llama
# or simply
promptfoo eval
```

Afterwards, you can view the results by running `promptfoo view`
